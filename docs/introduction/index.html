Introduction

Hi! This is Hoogii Wallet Developer Documentation. Hoogii commits to creating the most practical and easy-to-use crypto wallet on Chia Blockchain. With Hoogii Wallet, users will be able to easily manage your assets on Chia Network, and developers will be able to easily access the blockchain via the wallet and its companying software suites.

Reading through this documentation, you will learn how to build a dApp and connect with Hoogii wallet.

  • To get started, install Hoogii Wallet on Chrome Web Store.
  • For the source code, refer to our open source GitHub repository.
  • To receive the latest news, follow our Twitter, join our Discord, or read our Medium.

If further help is needed, please feel free to contact us at: contact@hoogii.app.
